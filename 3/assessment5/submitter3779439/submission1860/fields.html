Q3. The Economist article “Cancer’s epicentre” describes several drugs that affect epigenetic processes. Explain how Decitabine may be used to treat cancer, with reference to effects on the epigenome. In your answer include the following points:

  • Identify the class of epigenetic inhibitors that Decitabine belongs to. 
  • Describe the impact of Decitabine on DNA methylation.
  • Describe how Decitabine can have an anti-tumour effect. 

Decitabine is a type of hypomethylating agent that can inhibit DNA methyltranferase. It belongs to the demthylating or hypomethylating class of agents. It is thought to be a type of anti-metabolite which means that it helps to rid of abnormal MDS cells. The main goal of this medicine is to treat myelodysplastic syndrome, and it does so by producing normal blood cells while killing the aberrrant ones. When there is an increase in DNA methylation, there is an increase in blockage of the activity of the surpressor genes. These genes regulate cell division and growth, so when they are blocked the cells start to over produce forming cancer. This drug uses demethylation to restore normal function to tumor suppressor genes. When these tumor suppressor genes are restored, it can have an anti-tumor effect.
